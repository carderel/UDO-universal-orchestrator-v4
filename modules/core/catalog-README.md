# Project Catalog

Complete documentation history enabling LLM switching and full audit trail.

## For New LLMs

1. Read `../START_HERE.md` first
2. Then `../ORCHESTRATOR.md`
3. Check `../HARD_STOPS.md`
4. Check `sessions/` for latest session log
5. **Check `handoffs/` for any pending handoffs**
6. Continue where the previous AI left off

## Folders

- `sessions/` - **START HERE** - Session handoff documents from previous AI
- `handoffs/` - **CHECK THIS** - Agent-to-agent handoffs that may be pending
- `agents/` - Per-agent work logs
- `decisions/` - Key decisions with rationale
- `errors/` - Error tracking
- `archive/` - Compressed completed phases

## Resume Checklist

Before starting work:
- [ ] Read latest file in `sessions/`
- [ ] Check `handoffs/` for anything marked incomplete
- [ ] Review `../PROJECT_STATE.json` for current todos
