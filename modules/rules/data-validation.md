# Data Validation Rules

## Before Processing
- Verify file encoding
- Check for header row
- Note any missing values
- Validate data types

## During Processing
- Handle nulls/missing data explicitly
- Log anomalies
- Preserve original data

## Before Submitting
- [ ] Output format matches requirements
- [ ] Numbers verified
- [ ] Sources documented
- [ ] Confidence level stated
